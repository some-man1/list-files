# list-files
A C script can shows you the file in any dir on your PC
# Usage
## Step one(compiling)
## windows 
```shell
gcc lf.c -o lf.exe
```
## Linux
```shell
gcc lf.c -o lf
```
![Screenshot3](https://github.com/some-man1/list-files/assets/142589483/20782620-df99-4d69-9ae8-1665444b13d3)


## Step two(run it)
```Shell
lf /path/exmple
```
