# list-files
A C script can shows you the file in any dir on your PC
# Usage
## Step one(compiling)
## windows 
```shell
gcc lf.c -o lf.exe
```
## Linux
```shell
gcc lf.c -o lf
```
## Step two(run it)
```Shell
lf /path/exmple
```
## or if you did not write any argment it will show you the files in the working dir
```
lf
```
